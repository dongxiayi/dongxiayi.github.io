前言

由于本人前端和后端的代码基础不扎实，所以就用了最简单的github+hexo搭建网站

Github注册+登录方法

文章目录

  • 1.github介绍
  • 2.使用步骤
  • 3.接收学长发过来的通知

1.github介绍

  • 先说一下github是个什么东西，这是一个分布式管理平台，专门用来让程序员托管代码的，而现在我们要来完成如下几件事：
  • 1.注册
  • 2.验证登录

2.使用步骤

  • 我们首先需要注册这个github号，进入如下网址：
  • 点击此处进入github官网
  • 进入看到的是如上页面。接下来点击sign up
  • 进来之后就开始注册你的信息了 建议这里输入自己的qq邮箱和自己的密码 用户名最好是英文名字
  • 在这里选择左边这个免费的
  • 点击进入qq邮箱：点击此处进入qq邮箱 进来之后用你们的qq直接登录就可以了
  • 进入邮箱的收件箱点击github发给你的消息，以此确定完成了你的github注册功能！！！
  • 我们现在回到页面来进行登录我们的github了
  • 看到这个界面就代表你的github登录成功

仓库搭建

我们为什么要搭建仓库，我们的内容是不是要装到一个远程的服务器，所以我们就要利用github上面的仓库作为我们的远程服务器(白嫖党万岁!).然后将hexo博客的内容装到这个github的服务器上

搭建过程

  • 1.在github的页面上找到左侧的new按钮(一个绿色的)
  • 2.我们要一定记住我们的github用户名、注册的邮箱和密码，后面会用有
  • 3.这里填写的Repository name一定要填你注册github时的用户名，然后后面同意加上.github.io的后缀地址，之后再选择下面的Add a README file选项 这是一个阅读的文件
  • 4.显示这样的页面就代表建设成功了

SSH keys密钥生成

建好仓库之后我们需要建立一个关联的工作，将本地和github进行一个绑定，这个时候就需要用SSH

配置过程

  • 1.我们进到自己电脑的任意一个文件夹右击空白处，选择Git bash here
  • 2.我们在输入一个ssh看看有无建立，因为之前在搭建git的时候自己带了有一个ssh
  • 3.生成ssh: ssh-keygen -t rsa -C"邮件地址 这是我们注册时的qq邮箱地址" 这里出现选项直接按4次回车就行
  • 4.之后就在我们本地建立了一个ssh的文件夹(这个文件夹在c盘\用户\Administortar或者user 这里我电脑是ASUS)
  • 5.右击id_rsa.pub这个文件，选择NotePad++打开(这里我是安装了的 所以大家得自己安装notpad),之后再ctrl+A全选复制下来
  • 6.在打开我们的github，选择我们的setting选项
  • 7.选择ssh and GPG keys 在点击右边的new ssh key
  • 8.这里面的title你可以随便起名字，但是这个key一定是刚刚复制下来的rsa
  • 9.这样就代表安装成功了
  • 10.然后在确定ssh是否绑定成功: ssh -T git@github.com 这里就原封不动的复制下来
  • 这里会显示这个 输入yes就OK了 因为我这里安装过 所以没有出现，出现sucessful就代表成功了
